## 📖 Overview
**Afro-Arab Architect** is a comprehensive B2B supply chain ecosystem designed to streamline interactions between Admins, Regional Agents, Suppliers, and Associate Partners. It provides a robust 12-stage trade pipeline, real-time notifications, and professional document management.

- **Frontend:** React 19 + Vite 6 (TypeScript)
- **Backend:** Django + PostgreSQL
- **Mobile:** Capacitor 8 (Cross-platform)
- **Live Status:** Production VPS + Play Store (v4)

---

## 🚀 Key Features
- **12-Stage Trade Pipeline:** Track orders from Quotation to Commission.
- **RBAC:** Secure Role-Based Access Control for all stakeholders.
- **Real-time Notifications:** Instant updates on status changes.
- **Buyer Anonymization:** Professional privacy-first lead management.
- **Mobile Optimized:** Full native experience on Android and iOS.

## 🛠️ Local Development

### 1. Prerequisites
- Node.js (v20+)
- Python (v3.12+)
- PostgreSQL

### 2. Frontend Setup
```bash
cd frontend
npm install
npm run dev
```

### 3. Backend Setup
```bash
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python manage.py runserver
```
